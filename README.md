# Tableau Superstore USA Analytics Dashboard

This project is an end-to-end Tableau analytics report built using the Sample Superstore dataset.

## Assignments Covered

- Cross Tab with Percent of Total
- Inner Join Analysis (Netflix Dataset)
- Histogram (Customer Order Frequency)
- Top 50 Customers vs Others (Set)
- KPI Card with Growth Indicator (▲ ▼)
- Parameter-Based Dynamic Sales View
- Product Grouping with Avg Discount
- Fully Interactive Dashboard

## Dashboard Features

- Metric Parameter (Sales / Profit / # Orders)
- KPI Cards (Profit Margin & Return Rate)
- Filter Actions
- Show / Hide Filter Button
- Top 5 Products (Dynamic)
- Trend Lines & Average Lines
- Navigation Button

## Tool Used
- Tableau Desktop

## Dataset
- Sample Superstore Dataset
